# Diabetes Prediction (Pima Indians)

Short description:
This repository contains a small end-to-end data science project that predicts diabetes using the Pima Indians dataset. It demonstrates EDA, preprocessing, baseline and improved models, evaluation, and explainability.

## Files
- `data/pima_diabetes.csv` — dataset (or instructions to download)
- `notebooks/01-diabetes-analysis.ipynb` — walkthrough notebook
- `src/` — small helper scripts (data_processing.py, model.py)
- `app.py` — simple Streamlit demo (optional)

## How to run
1. Create a virtual env: `python -m venv venv && source venv/bin/activate`
2. Install: `pip install -r requirements.txt`
3. Open notebook: `jupyter notebook notebooks/01-diabetes-analysis.ipynb`
